# CastleGate - Virtually Me

En Central Hub för total kontroll över ditt digitala liv.

## Vision
"Vår vision är att ge alla människor möjlighet att ta kontrollen av sitt digitala liv"

## Projekt Översikt

CastleGate är en integrerad lösning där människor får full kontroll över:
- Dokument, data, egendomar och tillgångar
- Styr kommunikation
- Egen marknadsplats med valfrihet
- Säker blockchain-baserad datalagring

## Teknisk Stack

### Frontend (Mobil)
- React Native (iOS/Android)
- BankID integration för Norden
- Biometric authentication
- Cross-platform kompatibilitet

### Backend
- Node.js med Express
- Blockchain integration (Smart Contracts)
- AI/ML för GAI-assistent
- Secure storage solutions

### Blockchain
- Private Key management
- Smart contracts med Biometrics
- Castlegate Coins
- Content blockchain

### Web (B2B)
- React/Next.js
- Enterprise dashboard
- Permission marketing tools

## Projekt Fas

1. **Fas 1 - Data** (15 MSEK)
   - Private Key management
   - Egen säker box
   - Dokument/tillgångar management

2. **Fas 2 - Kontroll** (50-100 MSEK)
   - Total kontroll över data och tillgångar
   - BankID inloggning
   - Biometric security

3. **Fas 3 - Reklam**
   - Permission marketing
   - Användarstyrd reklam
   - AI-driven produktjämförelse

4. **Fas 4 - Marknadsplats**
   - Personlig marknadsplats
   - Castlegate Coins
   - B2B/B2C integration

## Installation

```bash
# Installera dependencies
npm install

# Starta utvecklingsserver
npm run dev

# Build för produktion
npm run build
```

Se `.env.example` för nödvändiga miljövariabler.
