# Castlegate B2C - Webbapplikation

En webbaserad demo-applikation för Castlegate B2C, byggd som förberedelse för mobilapplikationen.

## Teknologi

- **React 18** med **TypeScript**
- **Vite** för build och utveckling
- **Tailwind CSS** för styling
- **React Router** för navigation

## Installation

```bash
npm install
```

## Utveckling

Starta utvecklingsservern:

```bash
npm run dev
```

Appen körs på `http://localhost:5173`

## Bygg

Bygg för produktion:

```bash
npm run build
```

## Sidor

- **Hem** (`/`) - Översikt och snabbåtkomst
- **Konton** (`/accounts`) - Visa alla konton
- **Erbjudanden** (`/offers`) - Personliga erbjudanden
- **Brevlåda** (`/mailbox`) - Meddelanden och fakturor
- **Profil** (`/profile`) - Användarprofil och inställningar
